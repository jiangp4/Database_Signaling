In the search module, you can query either a cell signal molecule (e.g., TGFB) or a target gene (e.g., SOX4). With a cell signal molecule as the input, the platform will report the log2 fold change (logFC) of target genes upon treatments from that signal in all collected studies. Please use the auto-complete function in the input box for available signal names. With a target gene as the input, the platform will report the logFC upon the treatment of all cell signals.
