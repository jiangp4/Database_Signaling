Human cytokine response data

data matrix of cytokine treatment response: Each column represents a cytokine treatment experiment on a cell model under certain conditions. Each row represents a target gene. Each value is a log2 fold change between treatment and control conditions.

meta information of each treatment profile: Each column represents an experimental condition. Each row represents a profile ID used as column names in the data matrix above.

source code to filter response profiles: When building the CytoSig model, a critical issue is to filter cytokine treatment response profiles that are relevant to human physiological conditions. Here is the source code to compute quality measures as correlation between cytokine ligand or receptor expression and target gene expression.
